Nova Bridge is an AI-powered assistive platform designed to enhance the independence and quality of life for specially-abled individuals. It combines voice recognition, emotion detection, and intelligent automation to provide personalized support and real-time monitoring.

Key Features

Voice-Controlled Assistance – Users can interact with the system using natural voice commands to control devices, access information, or trigger tasks.

Emotion Recognition – Real-time analysis of facial expressions and speech patterns to detect emotional states and provide appropriate responses.

Intelligent Automation – Automates daily routines and tasks to reduce dependency on manual intervention.

Caregiver Monitoring Dashboard – Provides caregivers with a live overview of user activities, health status, and alerts for abnormal behavior.

Multi-Modal Support – Integrates vision, voice, and dashboard modules for a holistic user experience.

Secure & Configurable – Supports secure API key management and configurable backend services for AWS, Bedrock, and Nova Act AI engines.

Goal

To empower specially-abled individuals to perform daily tasks independently while giving caregivers actionable insights, ensuring safety, and enhancing overall wellbeing.
